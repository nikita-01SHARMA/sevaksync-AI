Description

SevakSync AI is a smart resource allocation platform designed for NGOs and community groups. It helps identify urgent local needs, manage volunteers, track resources, and improve coordination through data-driven decisions.

Features
Smart volunteer-task matching based on skill and location
Community needs management with filters and export options
Volunteer registration, history, and review system
Real-time dashboard with analytics and reports
Resource tracking with alerts for shortages
Interactive map for needs, volunteers, and resources
Dark mode and multilingual support
Tech Stack
React.js
Tailwind CSS
Node.js
Express.js
MongoDB / Local Storage
Recharts
Leaflet Maps
